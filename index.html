Carers & Seniors Discounts QLD

If you are a carer or senior within Queensland, you can use this search to help find businesses that offer discount products or services.

Search just like Google, with terms like physio, your local postcode, the name of a business.

Don't have a card?

  • Apply for a Carer Card
  • Apply for a Seniors Card

eg. cafe, hearing aid, 4000

E-mail for help!

This is a free to use service. Open data by data.qld.gov.au. You can find this project on GitHub - feel free to contribute!
